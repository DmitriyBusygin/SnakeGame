Учебный проект по написанию простой 2D игры змейка. 
Используемые технологии: Java SE, Swing.
1. Управление: W - вверх, S - вниз A - влево, D - вправо.
2. Змейка должка есть яблоки(красного цвета) и расти в длину.
3. При достижении конца игрового поля, змейка появляется на противоположной стороне.
4. Змейка может "откусить" своё тело.
5. Игра бесконечна.
